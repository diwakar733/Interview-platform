# Interview Platform Frontend

React + Vite frontend for the real-time interview platform.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```

## Features

- User Authentication (Login/Register)
- Dashboard with Interview History
- Create Interview Rooms
- Real-time Video Communication (WebRTC)
- Shared Code Editor (Monaco)
- Interview Timer
- Rating and Feedback System
